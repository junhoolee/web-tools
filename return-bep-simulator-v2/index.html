마진 설정

판매가 (Price)

의미 판매 단위당 수익. G(이익)과 가격 탄력성 모델의 기준점으로 사용됩니다.

영향 단위 이익(G) 직접 변동 / BEP 변동 / 가격 탄력성 통해 판매량 변동 / 규모효과 연쇄 변동

근거 Gentle Monster 등 프리미엄 아이웨어 가격대. $550은 탄력성 공식의 기준가로도 사용.

비유 택시 미터기처럼, 요금을 올리면 주머니엔 더 들어오지만 손님이 다른 앱을 켜기 시작한다.

매출원가 (COGS)

의미 제품 한 개당 원가. G(이익)을 줄이면서 동시에 L(반품 손실)을 늘리는 이중 타격 구조입니다.

영향 단위 이익(G↓) / 반품 손실(L↑) / BEP 변동 — 하나의 변수가 양쪽에 동시 작용

근거 $220은 판매가 $550 대비 ~60% 매출총이익률. 프리미엄 아이웨어 업계 평균 마진 구조에 부합.

비유 댐의 균열처럼, 원가가 오르면 마진에서도 새고 반품 손실에서도 동시에 새어나간다.

반품 비용

배송비 — 왕복 (Round-trip Shipping)

의미 반품 1건당 왕복 운송비. 소비자에게 반품 레이블을 제공하고 창고까지 회수하는 비용.

영향 반품 손실(L) 직접 증가 / BEP 하락 (더 낮은 반품률에서 적자 전환)

근거 미국/캐나다 UPS/FedEx 반품 레이블 편도 $12~18 합산. 국제 프리미엄 패키지 왕복 운임 기준.

비유 헛걸음 택배처럼, 물건이 돌아오는 순간 배달기사가 두 번 움직인 기름값을 내가 떠안는다.

검수/재입고비 (Inspection & Restocking)

의미 반품 제품의 상태 확인, 분류, 재포장, 재고 시스템 업데이트 인건비.

영향 반품 손실(L) 직접 증가 / BEP 하락

근거 3PL 창고 반품 처리 단가 $15~25. 아이웨어는 렌즈 정밀 검수가 필요하여 $20 설정.

비유 공항 입국심사처럼, 반품 상품도 재입고 전 '신원 확인'을 거쳐야 하고 그 시급이 원가에 얹힌다.

포장재 손실 (Packaging Loss)

의미 개봉 시 손상/재사용 불가된 포장재(박스, 봉인 씰, 완충재) 원가 손실.

영향 반품 손실(L) 직접 증가 / BEP 하락

근거 프리미엄 아이웨어 고급 패키징(자석 클로저 박스, 로고 각인 케이스) 단가 $10.

비유 생일 케이크 상자처럼, 한 번 뜯고 나면 다시 그 설레는 외형으로 돌아올 수 없다.

잔존가치 (Salvage Value)

의미 반품 제품에서 회수 가능한 금액. 재판매·리퍼브·부품 활용으로 L을 줄여줍니다.

영향 반품 손실(L) 감소 / BEP 상승(유리) / β 규모효과와 결합하여 판매량 따라 변동

근거 개봉 아이웨어는 위생·규제상 완전 회수가 어려움. B-Stock 중고 재판매 한계를 반영한 보수적 $10.

비유 중고 교과서처럼, 다 쓴 것 같아도 후배에게 팔면 커피 한 잔 값은 건진다.

반품 모델 (Weibull)

R∞ 최대 반품률 (Asymptotic Return Rate)

의미 시간이 무한히 흐를 때 이론적 최대 반품률. Weibull CDF의 점근선(Asymptote).

영향 각 시점 반품률 변동 / λ 역산 / 곡선 전체 형태 변경

근거 아이웨어 28%. 일반 패션(30~40%)보다 낮고 전자기기(10~15%)보다 높은 수준.

비유 수조의 만수위선처럼, R∞는 반품이 차오를 수 있는 최대 수위다. 수위가 높을수록 BEP를 넘기기 쉬워진다.

k 형상 파라미터 (Weibull Shape)

의미 반품의 시간 집중도. k>1이면 초반 집중(결함 중심), k<1이면 장기 분산(패션/핏 중심).

영향 반품 시점 분포 형태 / 특정 기간 내 누적률 변동 / λ 역산값 변동

근거 아이웨어 k=1.1. 전자기기(1.5, 초기 결함 집중)와 패션(0.85, 장기 분산) 사이.

비유 버스 파업 뉴스처럼, k가 크면 당일 민원 폭주 후 잠잠, k가 작으면 며칠에 걸쳐 드문드문 불평.

r14 기준 반품률 (14-day Observed Rate)

의미 구매 후 14일 시점의 실측 누적 반품률. R∞·k와 함께 λ를 역산하는 핵심 관측값.

영향 Weibull λ 역산 / 반품 곡선 속도 / 각 기간 내 예상 반품 비율

근거 20% = R∞(28%)의 ~71%가 14일 내 발생. 아이웨어 소비자 초기 2주 피팅 판단 경향 반영.

비유 검진 중간 결과지처럼, 14일 차 수치 하나로 전체 반품 곡선의 '속도'를 역추적하는 단서.

판매량 모델 (탄력성)

기준 판매량 (Base Volume)

의미 참조 조건(판매가 $550, 반품기간 30일)에서의 월 판매량. 탄력성 모델의 기준점.

영향 V_adj 기준점 / 규모효과(α·β) 분모 / 총 이익·총 반품 손실 절대값 변동

근거 단일 아이웨어 SKU 프리미엄 e-커머스 월 판매량 1,000개. 비율 계산의 정규화 기준점.

비유 체중계의 '표준 체중'처럼, 모든 변화를 이 기준 대비로 환산한다. 기준을 낮게 잡으면 규모효과가 크게 작동.

반품허용기간 (Return Window)

의미 소비자가 반품 가능한 정책 기간. (Window/30)^εw로 판매량에 영향.

영향 탄력성(εw) 통한 V_adj 변동 / Weibull 기간 내 반품 비율 / 총 반품 건수 변동

근거 30일 = e-커머스 업계 가장 보편적 반품 정책. Amazon, Shopify 기반 브랜드 표준.

비유 수영장 무료 체험처럼, 이용 기간이 짧으면 망설이다 등록을 안 한다. 고객의 '안전망' 크기가 구매 용기와 직결.

εp 가격 탄력성 (Price Elasticity)

의미 가격 변화 대비 수요 변화 민감도. 음수 = 가격↑ 수요↓. |εp|>1이면 탄력적.

영향 가격 변경 시 V_adj 변동 / 총이익·반품비용 연쇄 변동 / 규모효과 연쇄

근거 -1.2 = Kalla(2017) 패션 탄력성 연구, McKinsey 프리미엄 분석 종합. 명품보다 탄력적, 생필품보다 비탄력적.

비유 고무줄에 묶인 돌멩이처럼, 가격을 올리면 수요가 더 크게 반대로 튕겨나간다.

εw 반품기간 탄력성 (Window Elasticity)

의미 반품기간 변화 대비 판매량 민감도. 양수 = 기간↑ 판매↑ (소비자 신뢰 효과).

영향 반품기간 변경 시 V_adj 변동 / 총이익·반품비용 연쇄

근거 0.20 = Ertekin & Agrawal(2021, Management Science) 주얼리 소매업체 실험에서 역산.

비유 온천 온도처럼, 조금 더 따뜻하면 손님이 살짝 더 오지만 주된 이유는 아니다. 온화한 영향.

α 비용 규모효과 (Cost Scale Effect)

의미 판매량 변화 대비 단위 원가 변화. 음수 = 물량↑ 단가↓ (규모의 경제).

영향 V_adj 변동 시 조정 단위 원가 변동 / 총이익 개선 / BEP 변동

근거 -0.25 = BCG 경험곡선(80~85% 학습률) + 3PL 물량 할인 데이터 혼합.

비유 대량 구매 할인처럼, 물량이 늘면 원재료뿐 아니라 배송·포장·인건비까지 한꺼번에 단가가 깎인다.

β 잔존가치 규모효과 (Salvage Scale Effect)

의미 판매량 변화 대비 잔존가치 회수율 변화. 양수 = 물량↑ 회수율↑.

영향 V_adj 변동 시 조정 잔존가치 변동 / 반품 손실(L) 차감액 변동 / BEP 변동

근거 0.15 = B-Stock Solutions, Optoro 리커머스 데이터 종합. 벌크 재판매 협상력 + 전용 리퍼브 채널 효과.

비유 중고차 딜러 경매처럼, 매물이 많으면 경매장이 활성화되어 입찰 경쟁이 붙고 대당 낙찰가가 올라간다.
